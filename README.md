# SIMPE P2P - UDP HOLE PUNCHING

## How to use ?

To run the application, simply execute the following steps:
- Install dependency
  ```
  sudo apt update
  sudo apt install libmosquitto-dev
  ```
 - Build
   ```bash
   make
   ```
The program can be run with one **Publisher** and multiple **Subscirber**
